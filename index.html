Privacy Tools

Ubuntu Operating System

Ubuntu is an operating system based on Debian-Linux. It is easy to use and definetley a way more privacy friendly solution than windows. Also it is one of the most common Linux distributions. There is a lot of FOSS (Free and open source software) availible for it. Ubuntu itself is FOSS.

Cryptomator Storage

Cryptomator is an encryption tool. It's key feature is its cloud folder encryption. Basically you can encrypt a folder in one of many cloud services and acces the data even on mobile devices. It is available for Windows, Linux, MAC-OS, IOS and Android. Cryptomator is FOSS.

More
Nextcloud Storage

Nextcloud is FOSS which is basically a self hosted cloud. It's not very convenient to use. But it gets the job done. It supports multiple APPs written by the community. By default the data isn't encrypted (only on transfer). But you can enable server site encryption.

More
Matrix Chat

Matrix is a decentral chat service. It supports voice and video calls, voice messages, images and text. But there are no real limitations. There are many clients beacause it is FOSS. You can host your own server. But this server can communicate with all other servers as well as the users.

More
Signal Chat

Signal is a privacy friendly chat app compareable with WhatsApp. It also has the ability to send MOB (MobileCoins) to another user of the app. Also it has nearly all features off WhatsApp. After matrix it is the best chat service. It's partly open source.

More
KeePassXC Passoword Manager

KeePassXC is a password manager based on KeePass. It is crossplatform and FOSS. If youre tired of remembering passwords just give it a try. It's design is state of the art and it has a lot of features. For example plugins in the common browsers.

PiHole AdBlocker / DNS

PiHole is a network wide AdBlocker. But you can also block what ever you want. For example domains which are known for scamming, phishing or other stuff you want to protect your network from. You can also create groups to block content for only a few devices. It is FOSS.

More
PiVPN VPN

A self hosted vpn connection to your home network. So you can use your internal resources, turn on your pc per wake on lan and connect to it with RDP from all over the world. You can also use your PiHole. It is FOSS.

More
Firefox Browser

Firefox is a privacy fiendly browser with a lot of features such as plugins. Firefox is maintained by the non profit organization mozilla. Its apps run on every common operating system. You wont be missing anything.

More
DuckDuckGo Search Engine

A privacy friendly search engine. It's search results are similiar to the results of Google. So you can change to DuckDuckGo without any effort. DuckDuckGo does make money with ads to maintain the service. It has a lot of features just like safe search.

More
Protonmail E-Mail

Protonmail is a secure e-mail service located in switzerland. Protonmail encrypts all the emails. There were some issues where protonmail logged for the government of switzerland but proton fought for their customers.

More
Tor Browser / VPN

Tor is a complex network of vpn/proxy like connections. Your traffic goes threw an entry node. After that it will go to a few middle nodes. And at the end the traffic leaves the tor network by an exit node. The protocol tor is using is the onion protocol. Tor stands for the onion router.

More
Element Matrix Client Chat

Element is the most common matrix client. It has tons of features and is very similiar to discord. There are apps for a lot of common operating systems including mobile and web. There is a paid plan which offers to communicate with other chat apps in the element app. Their normal version is free.

More
Fluffy Chat Matrix Client Chat

Fluffy chat is another chat client for matrix. It looks much more like signal and WhatsApp. The only disadvantage is the missing support of calls. But this may be updated in the future.
